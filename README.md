# Tech Stack
- Programming Language: JavaScript (Node.js)
- Runtime: Node.js
- Framework: Express.js
- Database: MongoDB / Mongoose (or MySQL / Sequelize)
- Authentication: JSON Web Token (JWT)
- Environment Management: dotenv
- Testing: Jest / Mocha (optional)
